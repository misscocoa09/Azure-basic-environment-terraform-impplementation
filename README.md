# Azure-basic-environment-terraform-impplementation
To execute this project with Terraforrm and replicate it, I used an Azure pay as you go subscription. I downloaded theazure commands to use azure rm commands with terraaform because without it you cannot initialize in your environment or authenticate. Make sure to do this firtst of all. Then proceed with:
1. Visual Studio Code
2. Terraform extension. Once downloaded in your VS code, please enable it by creating a new file and naming it .tf at the end. example: main.tf.
3. copy myy script showing in the main.tf file of this git and paste it.
4. execute in the VS bash: terraform init, then, terraform plan , then terraform apply. That is it! :)
